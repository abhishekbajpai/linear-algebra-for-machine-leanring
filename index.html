Types of Vector

Zero Vector, are vectors composed of zeros, and zero only. All entries of these vectors are denoted by 0. v = [0, 0, 0]
Unit Vector, are vectors composed of single element equal to one, and rest to zero. All entries of these vectors are denoted by 0.
v1 = [0, 1, 0] v2 = [0, 0, 1] v3 = [1, 0, 0]
