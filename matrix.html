Matrices

A matrix is a rectangular array of numbers, symbols and expressions which are arranged in rows and columns. The numbers, symbols, or expressions in the matrix are called its entries or its elements

$\overrightarrow{v} = \begin{bmatrix}1 & 0 \\0 & 0 \end{bmatrix}$

Types of Matrices

Square Matrix, A matrix is said be square matrix if number of rows and columns are equal. Square matrix are sometimes used to represents a linear transformation from a vector space to itself such as Reflection, rotation and shearing The given matrix is of 3 * 3 which means this matrix have 3 rows and 3 columns
$\begin{bmatrix}1 & 2 & 3 \\4 & 5 & 6 \\7 & 8 & 9 \end{bmatrix}$
Rectangle Matrix, If the number of rows in a matrix is not equal to the number of columns in it then the matrix is known as a rectangular matrix. The given matrix is of 2 * 3 which means this matrix have 2 rows and 3 columns.
$\begin{bmatrix}1 & 2 \\4 & 5 \\7 & 8 \end{bmatrix}$
Diagonal Matrix, is a square matrix in which all of the elements except the principal diagonal elements are zeroes.
$\begin{bmatrix}1 & 0 & 0 \\0 & 5 & 0 \\0 & 0 & 9 \end{bmatrix}$
Scalar Matrix, is a square matrix in which all of the principal diagonal elements are equal and the remaining elements are zero. It is a special case of a diagonal matrix and can be obtained when an identity matrix is multiplied by a constant numeric value.
$\begin{bmatrix}2 & 0 & 0 \\0 & 2 & 0 \\0 & 0 & 2 \end{bmatrix}$
Identity Matrix, is a square matrix whose principal diagonal elements are ones, and the rest of the elements of the matrix are zeros. An identity matrix is always a square matrix and is expressed as “I.” Identity Matrix is also known as Unit Matrix.
$\begin{bmatrix}1 & 0 & 0 \\0 & 1 & 0 \\0 & 0 & 1 \end{bmatrix}$
Symmetric matrix, In linear algebra, a symmetric matrix is a square matrix that is equal to its transpose. A^{ \mathrm{ T } }
Zero Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Upper Triangle Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Lower Triangle Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
