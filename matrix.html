Types of Vector

Zero Vector, are vectors composed of zeros, and zero only. All entries of these vectors are denoted by 0.

$\overrightarrow{v} = \begin{bmatrix}0 & 0 \\0 & 0 \end{bmatrix}$

Unit Vector, are vectors composed of single element equal to one, and rest to zero. All entries of these vectors are denoted by 0.

$\overrightarrow{v1} = \begin{bmatrix}0 \\1 \\0 \end{bmatrix}$

$\overrightarrow{v2} = \begin{bmatrix}0 \\0 \\1 \end{bmatrix}$

$\overrightarrow{v3} = \begin{bmatrix}1 \\0 \\0 \end{bmatrix}$

Sparse Vector, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.

Vector Operations

vector-vector addition

Vector addition is element wise addition, and can only possible if both vectors are of same size i.e. Number of elements to be same. For Example: Let's say we have vector

$\overrightarrow{x} = \begin{bmatrix}1 \\2 \\3 \end{bmatrix}$
,  
$\overrightarrow{y} = \begin{bmatrix}4 \\5 \\6 \end{bmatrix}$
$\overrightarrow{x} + \overrightarrow{y} = \begin{bmatrix}1 + 4 \\2 + 5 \\3 + 6 \end{bmatrix} = \begin{bmatrix}5 \\7 \\9 \end{bmatrix}$

Main properties of vector addition

  • Associativity: (a + b) + c = a + (b + c)
  • Commutativity: a + b = b + a
vector-scalar multiplication

Vector scalar multiplication is element wise multiplication, Here we multiply each element with scalar.

For Example: Let's say we have vector $\overrightarrow{x} = \begin{bmatrix}1 \\2 \\3 \end{bmatrix}$ and scalar y = 5
$\overrightarrow{x}\cdot y = \begin{bmatrix}5*1 \\5*2 \\5*3 \end{bmatrix}$
$\overrightarrow{x}\cdot y = \begin{bmatrix}5 \\10 \\15 \end{bmatrix}$

Main properties of vector addition

  • Associativity: (a*b)*c = a*(b*c)
  • Commutativity: a + b = b + a
vector-vector dot product

This is one of important feature of matrix multiplication, where you will be given two vectors and then you will be asked for dot product of those.
