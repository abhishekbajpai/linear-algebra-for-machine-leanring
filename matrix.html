Matrices

A matrix is a rectangular array of numbers, symbols and expressions which are arranged in rows and columns. The numbers, symbols, or expressions in the matrix are called its entries or its elements

$\overrightarrow{v} = \begin{bmatrix}1 & 0 \\0 & 0 \end{bmatrix}$

Types of Matrices

Square Matrix, A matrix is said be square matrix if number of rows and columns are equal. Square matrix are sometimes used to represents a linear transformation from a vector space to itself such as Reflection, rotation and shearing
$\begin{bmatrix}1 & 2 & 3 \\4 & 5 & 6 \\7 & 8 & 9 \end{bmatrix}$
The given matrix is of 3*3 which means this matrix have 3 rows and 3 columns
Rectangle Matrix, are vectors composed of single element equal to one, and rest to zero. All entries of these vectors are denoted by 0.

$\overrightarrow{v1} = \begin{bmatrix}0 \\1 \\0 \end{bmatrix}$

$\overrightarrow{v2} = \begin{bmatrix}0 \\0 \\1 \end{bmatrix}$

$\overrightarrow{v3} = \begin{bmatrix}1 \\0 \\0 \end{bmatrix}$

Diagonal Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Scalar Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Identity Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Symmetric matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Zero Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Upper Triangle Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
Lower Triangle Matrix, most of the elements are zero, store only non-zero values and their corresponding indices, only the non-zero values are explicitly represented.
